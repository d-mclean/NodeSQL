# NodeSQL
Basic repo which connects to a SQL Azure instance using Node.

# Requirements
1.  An Azure SQL database
    (with a server-level firewall rule for the public IP address of the computer you use)
    
2.  Node.js
    (this example was done on Windows)
    
3.  npm init -y

    npm install tedious
    
    npm install async
    
4.  Run code
  (node <FILENAME>.js)
